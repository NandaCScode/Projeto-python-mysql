# 📝 Sistema de Lista de Tarefas com Python + MySQL

Este projeto é um sistema simples de gerenciamento de tarefas desenvolvido em Python com integração ao banco de dados MySQL.

O objetivo é praticar conceitos de backend, banco de dados e CRUD (Create, Read, Update, Delete).

---

##  Funcionalidades

- ✔ Adicionar tarefas
- ✔ Listar tarefas cadastradas
- ✔ Atualizar status das tarefas - pendente
- ✔ Remover tarefas -pendente
- ✔ Integração com banco de dados MySQL

---

##  Tecnologias utilizadas

- Python
- MySQL
- mysql-connector-python

---

## Aprendizados

Com este projeto, pratiquei:

- Conexão entre Python e banco de dados
- Comandos SQL (INSERT, SELECT, UPDATE, DELETE)
- Estrutura de CRUD
- Uso de cursor e commit
- Organização de código em Python
